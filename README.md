
# Project Title

implementation of optimizers and regularizations techniques and test the result of them in Fashion MNIST classifier neural network.

## Features

- Python
- Neural Networks
- Fashion MNIST
- Optimization 
- Regularization

## Optimizers

- SGD
- Momentum-SGD
- AdaGrad
- RSMprop 
- Adam

## Regularizations

- L1 Regularization
- L2 Regularization
